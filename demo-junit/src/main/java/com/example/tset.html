想轉行IT 有咩選擇? 一個學位、一個碩士都可以幫到你轉行。今期我地就主力介紹 IT Coding Bootcamp ，只需三至四個月就可以輕鬆轉型，有些公司更保證會幫你搵份好工！

市場上有大量 Coding Bootcamp 供人選擇，看到老眼昏花也未必找到適合自己的課程，甚至越看越亂遲遲下不了決定。這篇文章可以助你分析，哪類型的 IT Coding Bootcamp 最適合自己：

環球趨勢

IT 現時是一個環球性趨勢，事關全球各地不少生意都已經開始去實體化，就算沒有實體店存在，網上平台已經可以有效維持生意。疫症爆發其實也印證了這是鐵一般的事實，由於不少人在疫下被迫留在家中渡日和工作，令市場發現更多工序可以透過非實體途徑完成。例如購物方面，越來越多人選擇網購，令無店面或非實體店生意大增，相對的，開設一間實體店對於公司而言的重要性已逐漸降低。

美國市場研究公司電商市場（ eMarketer ) 今年年初發表報告指，2021年全球電商（ e-commerce） 市場銷售額將會突破5兆美元。統計處統計資料亦顯示，今年一月本港非實體零售銷售額為9.96億港元，比去年同期 6.47 億港元急升五成三。可見無論是本地還是全世界，去實體化的情況已經越來越普及。眼見電子市場有嚿「肥豬肉」，不少公司已實行電子化，逐步減少實體生意的人手需求，令一群人正面臨被行業淘汰的危機。如果要在社會繼續發展，跟隨著大趨勢而轉行做 IT 或是個不俗的選擇，而現時在職場上最多人會透過報讀 Coding bootcamp 入行。

微學位 (Bootcamp)  vs 學位 / 碩士

IT Coding Bootcamp 在外國一早已經很流行。近年香港亦有不少機構集合業界專才開辦 Coding Bootcamp。兩種方法都可以轉到行，到底微學位（Bootcamp）同學位實際上有哪些分別？

1. 學習時長

學位：需花上約四年時間完成

碩士：需花一年時間，兼讀需花兩年時間

微學位 Bootcamp：普遍需花三至四個月就可以完成，平均為期十三至十四個星期

2. 學術性 vs 實用性

學位：著重學術性，學習範圍較為全面，會學習電腦學（computing）、電腦科學（computer science）、用戶界面設計（UX/UI Design) 等實用性知識內外，課程亦學習更多理論性科目，時間最長但內容最全面，助你成為擁有全面知識的 IT達人。

碩士：學術性及實用性同樣著重，比學士學位更深層次的課程，助你可更深入認識一個行業，費用最高但認授性最強。

微學位 Bootcamp：著重實用性，針對業界現時需求需制定課程，以學習軟件告項目主導，會學習 Java, Java Script , AWS ,  GitLAB , API 等軟件。 內容並非最全面，但可以在短時間內成為一個 IT beginner，助你入行/轉行。性價比絕對比其他選擇為高。

3. 入讀門檻

學位：需要以學業成續作為基礎，主要透過考試制度 或者基礎 / 高級文憑 等渠道升讀

（包括：HKDSE/ IB / GCE / Higher Diploma/ Associate Degree 等）

碩士：需持有相關學科之學士學位

微學位 Bootcamp：零 IT 經驗的人士，甚至連文科生均可以報讀

甚麼 Coding Bootcamp 適合你？ 

全日制（Full-time） ? 兼讀制（Part-time） ? 遙距上課 ? 

讀一個IT Coding Bootcamp，一般至少要花上三個月時間，說的是全日制培訓課程，兼讀制或者網上遙距課程的話，就要花上更長時間。不過整體而言都比一個學位快捷。

揀選一個 Coding Bootcamp，最先要看外在條件。

1. 金錢、時間

普遍一個 Full-time IT Coding Bootcamp 都要用平日白天時間上堂，一星期上足五日，朝九晚六，等同上一個學期的課程。不過數個月裡每日沉浸在IT知識中，無論吸收力度和效率都會好得多。然而，一個 IT Coding Bootcamp 的學費並不便宜，上全日制課程的話一般需要 40000 – 60000元，除非你有份兼職，否則就等同零主動收入*。因此起碼確保自己有足夠生活五至六個月的資金才考慮上全日制。

*主動收入：需要付出勞動先得到的收入*

如果無辦法負擔到幾個月有出無入的風險，兼讀制或者網上遙距 IT Coding Bootcamp 會更加適合。學費一般為10K 至 80K 不等，上堂時間於周末或晚間。有些班會有 Hybrid course 可自由選擇每一堂上實體堂還是遙距。

除外在條件，到底想報讀的原因是甚麼？
入行？轉行？行內增值？純粹有興趣？

2. 知識需求量

每個人希望報讀 IT Coding Bootcamp 的原因都不相同。

如果你一心想轉行或者入行 IT 行業，其實沒有太多後顧之憂，全日制課堂必然是最周密和最全面，可以更快速助你裝備好自己投身 IT 界，例如 Full Stack Software Engineering 課程可以令你掌握多種投能，並獨立完成產品。

如果你希望行內增值，為爭取更多工作或晉升機會，工作經驗一定不能缺少，因此兼讀制或者遙距上堂會是個不俗的選擇。

如果你單純對 Coding 感興趣，又無迫切性的話，可以先考慮試讀一下網上平台上的免費 Coding course 或者 Bootcamp，可以先以自學模式鑽研一下。當你確定自己需要呢方面的技能，再去報讀正式課程亦未遲。

讀完Coding Bootcamp有何出路？

完成IT Coding Bootcamp 後，普遍入門級可以做到 Web Programmer。

其次，還有十幾個工種可以申請：

1. Application Developer

專門為人設計、創作、部署及更新特定設備/網站/特定網絡系統的軟件。

屬較為百搭的工種，皆因不同行業都需要人為他們更新軟件。

2. Analyst Programmer

專門為人確定、研發、測試、分析及維護軟件的應用程序。

工作包括編寫軟件、為軟件做編程及應用。

3. Back End Developer

專門建立、維持、測試及調適整個網站中瀏覽者不會見到的部分，

管理範圍包伺服器、應用程序及資料庫。

4. Development Operations (DevOps) Engineer 

專門彌合軟件開發及 IT 操作之間的差距，工作範圍介乎開發及部署程序之間，即是確保在軟件發佈或進入操作程段前可以消除障礙。

5. Digital Marketer

專門管理一間公司網頁、社交網站、搜尋器排行、電郵推廣、網上刊登ads及企業博客。

6. Front End Developer

專門負責對網頁前端，即瀏覽者可以見到的部分進行編碼，設計的不是網頁的外貌，而是其操作該設計如何在網頁上實現。

7. Full Stack Developer

專門把網頁前端(Front End) 及後端(Back End) 程式一併管理，要處理涉及資料庫、建立面向用戶的網站及要在項目的規劃階段與客戶合作。

8. Junior Developer

專門幫助開發團隊處理軟件設計及編碼的工作，學習代碼庫和收集客戶資料。

9. Product Manager

專門管理一個產品的生命周期及路線圖，主要工作是確認客戶需求及確立一個產品或功能可以實現的一個目標。

10. Software Engineer

專門應用數學分析和計算機科學原理來設計及開發計算機軟件。

11. Technical Support Specialist

專門為企業及消費者提供軟件及硬件上的援助或技術支援，例如系統連作援慢、連接問題、或無法顯示資料等問題。

不過約四個月的課程始終屬於入門級，多了一技旁身，若果多加鑽研攻讀一下其他課程，亦有機會找到 Mobile App、Machine learning 、Fintech 等工種。

一般公司會請 Coding bootcamp 畢業生？

根據 Linkedin 報告顯示，專業工程師（Specialized Engineers）為2021年最受歡迎行業之一。報告指出，在2019年至2020年間，行內職位需求增加25%，當中包含 Web Developer 在內。而 Project Management、Digital Marketing 和 Product Management 亦為被當局列為今年最受歡迎的 Hard Skills。

由於現時IT 行業是一個趨勢，市場上極度需要大量人才去發展這方面嘅技術。因此不少機構都正等待識 Coding 技術嘅人進駐。參考 Linkedin 網站的職位空缺欄目，以上工種現時共有約5600個職位空缺，其中 Data Analyst、Software Engineer 和 Product Manager 的空缺最多。而參考香港舉辦 Coding Bootcamp 機構的數據，畢業生的就業率平均為90%或以上。

雖然市場上有大量職位空缺，不過公司是否聘請也要視乎個人表現而定。就此，可以參考 讀完 IT Coding Bootcamp 一定搵到工?，了解企業對進修 IT Coding Bootcamp 對畢業生的要求。

如何選擇適合自己的機構？

1. 課程的技術領域分佈

選擇 IT Coding Bootcamp 之前，最好先了解清楚課程內容。其實 IT Coding Bootcamp 的學習領域都有好幾種，包括 JAVA, Javascript , React , API Development , GitLab, MTSQL 及 AWS 等。 不少市面上的機構都會教你用以上的軟件，令你最基本可以成為一個入門級 Web Developer。如果課程中有前端（Front end） 、後端（ Back end） 等專業領域技術 ，除了對行業會有更充分理解，亦可以跳到更多範疇的 IT 工種。不妨看看各機構的大鋼和留意 Coding Bootcamp 畢業生分享後再去分析。

2. 教學方式

事實上，不少讀 Coding Bootcamp 的人本身對數字並不敏感。特別是文科出身又想入行的人，要他們面對一堆數字，本身就是一個大挑戰 。所以，導師的教學方式往往決定你是否能夠享受一個課堂和引起學習興趣。

市面上的機構都會詳細列出導師出自哪間大學、有甚麼學位、在甚麼公司等。其實每間機構的導師在行內均擁有一定資歷，相當出色。

不過試想一下，在求學時期可能曾遇過學歷很高，但教書就「半桶水」的人，又或者教學方式不適合自己的老師，因此「有資歷」同「識唔識教」是兩碼子的事。要分辦一位導師「識唔識教」或者上堂氣氛好不好，不可以單靠客觀因素支持，只能依靠每個人的主觀感受判斷。加上 IT Coding Bootcamp 費用不便宜，如果希望在付出一筆錢之前，就提前了解清楚課堂是否適合自己，除了比較各機構的課程大鋼 ，不妨花些時間去報讀各公司的試堂（Trial class） ，早點感受上堂氣氛是否適合自己，萬一不適合亦尚有機會「轉呔」。

3. 支援機構的配套

舉辦 Boot Camp 的公司本身和業界都有一定聯繫，選擇時可以留意機構與哪些公司有聯繫。有心儀公司的話，亦可以以此為選擇機構的因素。在香港有三個主要機構會舉辦 Coding Bootcamp，包括 

Xccelerate、Tecky Academy 和 Venturenix LAB。這些機構平均就業率有90%或以上，而且平均畢業後都可以都找到 20K 或以上的薪酬。

1. 價錢參差

雖然 IT Coding Bootcamp 程價格參差，不過當中的差距不是想像般那麼大。例如較貴的課程中，亦會有分期付款的選擇，除開每月只需要供款數千元的。甚至可以留意某些機構會提供各式各樣的折扣優惠，例如提供 Early-bird 價，只要看準時機報讀，就可以節省一筆報讀課程的成本。另外，還有些機構亦會將價錢以付款方式劃分，如付現金或銀行轉賬可比信用卡享有更低價錢。

2. 就業支援為何如此重要？

幾乎間間 IT Coding Bootcamp 機構都會在簡介中寫明，有90%或上畢業生成功就業，即每10個人就有9個人成功就業。但試想象一下，萬一畢業後成為剩下未能入行就業的1人，花費這麼多金錢、報讀一個以轉行為目標的課程，豈不是徒勞無功？因此選擇一間機構前，除了格價，也需要留意機構如何支援無法就業的學員，如提供求職服務。

入讀準備事項

不少人在入讀 IT Coding Bootcamp 前都會問：「我入讀前要有甚麼準備？要事前讀很多理論預習嗎？」

答案是，不用。

如果要準備很多才可以入讀，基本上已有違 Bootcamp 的初衷。舉辦 IT Coding Bootcamp 的原意就是希望培訓零底子的人進入 IT 行業。若果真的想做事前準備，令自己放心入讀，反而需要做的是做好時間分配及調整心態。

1. 時間分配

四個月的 IT Coding Bootcamp 時間相對密集，平日一至五要上堂，周末要處理課業，因此要做定心理準備，報讀 IT Coding Bootcamp 後將減少自由活動時間。在課程開始前，需預先分配好時間，在課程與休閑時間之間取個平衡，例如確保一些與課程時間重疊的活動已完結或已安排至另一時段進行。就連周末也要排出部分時間，好讓自己有充足時間處理課業及消化過去一周的課堂內容。

2. 調整心態

因為要在四個月時間內面對自己完全不熟悉的範疇，至關重要的一點是保持思想開放。

一開始上堂，因為要開拓一個未知的領域，必然會覺得好艱難、好難上手和好辛苦。是痛苦的，但萬事起頭難，學習任何事物都必定要面對此階段。而根據我們學員的經驗，通常捱過首一至兩個月後，就可以過渡此階段，之後慢慢上手就會得到學習的滿足感，離成功之路越來越近。不過大前提是要保持思想開放，方能更有效地吸收知識。

所以讀書前要準備的，就是你的心態同時間，確保這四個月的時間是百分百準備好。大大提高轉行成功的機會。
